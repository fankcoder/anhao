
# 🚀 Vibe Coding 示例项目：用 AI 开发一个完整网站

> 本项目用于展示：如何通过 **Vibe Coding（氛围编程）** 的方式，借助 AI 从 0 到 1 构建一个真实可运行的网站应用。

----------

## 🎯 项目目标开发一个CS2玩家社区网站

**Vibe Coding = 用自然语言开发软件**

**CS2全球热门的FPS射击游戏**

本项目用于演示：

-   ✅ AI 如何参与真实网站开发
    
-   ✅ Prompt 如何一步步生成代码
    
-   ✅ 人类负责决策，AI 负责执行
    
-   ✅ Vibe Coding 的完整流程
    
-   ✅ 从需求到成品的全过程
    

项目不是一次性生成，而是多轮协作完成。

----------

## 🧩 项目功能

-   🌐 基于 Next.js App Router 架构
    
-   🎨 现代化 UI 风格
    
-   📦 API 接口路由
    
-   🧠 完全由 Prompt 驱动开发
    

----------

## 🧑‍💻 大致开发流程

###  让ai生成项目结构

示例 Prompt：

你是一个资深前端工程师。  
请帮我创建一个 Next.js App Router 项目，名称为cncs包含：  
1. 首页  
2. About 页面  
3. API 路由
4. ...

----------

### 让ai生成页面 UI

帮我写一个页面，包含：  
- 文本输入框  
- 提交按钮  
- 结果展示区域  
要求：现代 SaaS 风格 UI

----------

###  让ai优化界面统一风格

请把这个页面改成现代科技风：  
- 居中卡片布局  
- 圆角  
- 阴影  
- 按钮 hover 动效  
- 清爽字体

----------

### 让ai修复代码错误

这是我的代码和报错信息：  
<粘贴代码>  
  
请解释错误原因，并给出修改后的完整代码。

----------

## 📘 Vibe Coding 提示词公式

> **角色 + 目标 + 约束 + 风格**

示例：

你是一个高级前端工程师。  
请用 Next.js 写一个页面：  
- 有输入框  
- 有提交按钮  
- 风格是现代 SaaS 风

----------

## 🔁 迭代型 Prompt（非常重要）

让界面更简洁  
让布局更合理  
增加 loading 动画  
让代码更清晰  
优化组件拆分
